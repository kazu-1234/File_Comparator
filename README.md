# File Comparator (ファイル比較ツール)

指定された「元フォルダ」と「比較先フォルダ」を比較し、不足しているファイルや更新されたファイルを検出するツールです。
Google Drive（パソコン版ドライブ）などのクラウドストレージ上のファイルを、ダウンロードせずにメタデータのみで高速に比較する機能を備えています。
